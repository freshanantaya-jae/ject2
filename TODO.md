## 📋 1. สรุปความต้องการของระบบ (Requirements)

ระบบตรวจสอบคุณภาพแผ่นอะลูมิเนียมแบบไฮบริด โดยประยุกต์ใช้ **Deep Learning (YOLOv8n)** ร่วมกับ **Machine Learning (Decision Tree)**:
* **การรับภาพ:** สตรีมภาพสดและดึงภาพเฟรมสแกนผ่านกล้อง **USB Webcam**
* **สถาปัตยกรรมระบบไฮบริด (AI + ML):**
  1. **YOLOv8n (Deep Learning):** ทำหน้าที่ตรวจจับและวิเคราะห์ตำแหน่งวัตถุในภาพ ได้แก่ รอยขีดข่วนชำรุด (`scratch`) และรูเจาะขนาดเกลียวเมตริก (`M3`, `M4`, `M5`, `M6`)
  2. **Decision Tree (Machine Learning):** รับฟีเจอร์เด่น 7 มิติ (จำนวนชำรุด, จำนวนรูแยกตามมาตรฐาน, ระดับความมั่นใจเฉลี่ย, จำนวนวัตถุทั้งหมด) เพื่อใช้ในการแยกแยะพยากรณ์สถานะสุดท้ายเป็น **PASS (ผ่าน)** หรือ **FAIL (ชำรุด)**
* **การตัดสินคุณภาพ (QC Decision):** พยากรณ์ผลด้วยโมเดล Machine Learning ที่เรียนรู้และฝึกสอนด้วย Dataset แทนการเขียนเงื่อนไขด้วยมือ
* **การเตือนผ่านฮาร์ดแวร์บอร์ด (Hardware Signals):**
  * ชิ้นงานผ่าน (PASS): ไฟเขียวติด (RGB LED 4) + เสียงเตือนสั้น 1 ครั้ง (Buzzer)
  * ชิ้นงานชำรุด (FAIL): ไฟแดงติด (RGB LED 4 + LED 2, 3) + เสียงเตือนยาวติดกัน 3 ครั้ง (Buzzer)
* **การจัดเก็บข้อมูล:** บันทึกเวลา ผลสถานะ จำนวนรู ขนาดรูเจาะมาตรฐาน และเหตุผลตกเกณฑ์ลงในฐานข้อมูล **SQLite**
* **ส่วนแสดงผล:** เว็บ **Dashboard** (Glassmorphism Dark Mode) เพื่อเปิดดูสตรีมกล้องสด, กด Inspect, แสดงภาพตรวจจับ Bounding Box, แสดงรายงาน Features Vector ของ ML, แสดงสถิติรวม และตารางบันทึกประวัติย้อนหลัง
